# Árukészlet leltár program

Egyszerű Python program árukészlet nyilvántartására.

## Funkciók
- Termékek felvétele
- Lejárati dátum ellenőrzése
- Lejárt termékek listázása
- Bevásárlólista készítése
- CSV fájlba mentés

## Futtatás
A program Python 3.x verzióval futtatható.

```bash
python Raktarkezelo.py
